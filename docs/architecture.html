TargetKindHow 8kEdu hits it
Heartbeatclaw-agentOpenClaw Cron wakes on interval → reads Supabase state → advances curriculum or checks channels → acts or waits. Trigger is time/state, never a human message.
Autonomousclaw-agentFinds lectures, builds & repairs the curriculum, generates + verifies widgets, alerts on new uploads, recovers from interruption via persisted state — all unprompted.
Persistentclaw-agentSupabase = the agent's workspace/memory/session: learner model, course, mastery, run log. Literally the requirement.
NemoClaw + OpenShellbountynemoclaw onboard --agents agents.yaml. Agent has web (Apify/yt-dlp) + exec (runs generated Python) → genuinely worth containing. OpenShell allowlists youtube/apify/supabase + guards the filesystem. Demo: agent reaches an off-list host or ~/.ssh → blocked + logged.
Nemotronbountynemotron-3-nano-omni is the whole brain — one model reads video frames (omni vision), plans the course, generates widget specs, grades quizzes, calls tools. "Why": multimodal + thinking budget + tool-calling in one open, sandbox-runnable model.
vLLMbountyBlueprint vllm serving profile hosts Nemotron; local LM Studio proves the loop today.
SupabasesponsorTwo tiers: global content cache (compute once, reuse for all users) + per-user state. Postgres + realtime for the alert stream. The caching moat.
ApifysponsorDiscovers lectures for a goal and monitors channels for new uploads — the heartbeat's eyes on the world.
Most CommercializablebountyA personal AI tutor that auto-builds your course from YouTube — obvious market, near-zero marginal cost via the shared cache.
Recursive IntelligencetrackQuiz performance feeds back each cycle → the agent re-sequences the curriculum and re-picks sources. Self-improving.
Frontier factorjudgingCreativity: a contained, code-executing learning agent. Performance: global cache + small-active (A3B) model = fast + scales to many learners.

The loop

What runs every heartbeat

HEARTBEAT (OpenClaw Cron) — wake, read state from Supabase: JOB 1 · advance curriculum goal not met? gap in coverage → Apify: find a lecture for the weakest concept unprocessed video → process_video: frames → omni → widgets new material → Nemotron: re-sequence course, update mastery else → idle till next tick JOB 2 · monitor channels new upload since last cycle? → build its dashboard → alert learner every cycle → write run log + state (recoverable) quiz results → re-plan next cycle (recursive intelligence)
